A news application built with React Native that fetches the latest news from a public API, displays articles with images, allows users to view article details, and save favorite articles for later reading.

---

##  Features

- Fetch latest news from a remote API
- Display news list with images and titles
- Article details screen with partial content
- Open full article via external URL
- Add / remove articles from Favorites
- Favorites screen to view saved articles
- Persistent local storage for favorites

---

- React Native
- TypeScript
- React Navigation
- REST API integration
- AsyncStorage (local persistence)
- Functional components & React Hooks
